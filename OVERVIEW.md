An intelligent cover letter generation system leveraging LangChain and LangGraph to create personalized, professional cover letters tailored to specific job positions. The application parses .docx CVs and job descriptions, performs semantic skill matching using FAISS vector embeddings, and employs a ReAct agent powered by GPT-4o to generate contextually relevant letters in three distinct tones (professional, creative, technical). Features include semantic search over CV content for intelligent experience retrieval, automated skill-gap analysis with cosine similarity matching, and an interactive refinement chat interface for iterative improvement. Built with a focus on factual accuracy and grounded responses, ensuring all generated content is verifiable from the candidate's background.
